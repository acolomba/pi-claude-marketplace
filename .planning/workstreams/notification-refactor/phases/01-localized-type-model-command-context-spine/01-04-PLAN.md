<objective>
Migrate the mixed-subject cascade producers — the `import` command and the load-time
`reconcile` cascade (the `pending` command + the `reconcile-applied-cascade` emitted
by apply.ts/apply-outcomes.ts/notify.ts) — to the command-local notification model.
Each gets a co-located messaging module declaring its `CommandContext` (with
`Messaging.label`), its status set (the union of statuses it actually emits per the
RESEARCH inventory), and a render map total over that set (lifting the matching
renderPluginRow / renderMpHeader arms verbatim). Call sites thread notifyWithContext
with Plural cardinality annotation.

Purpose: Complete the D-13 full cutover — the mixed-subject cascades are the last
producers (not slash commands but cascade producers reusing the same per-status
shapes, RECON-04). Their render dispatch must also localize (MOD-01/03) so no command
remains pointing at the central switches, giving the Phase 4 ≤3-files proof a clean
baseline. Output-neutral.

Output: import's messaging module + a shared reconcile messaging module (pending +
applied-cascade) + call-site rewires across execute.ts / pending.ts / apply.ts /
apply-outcomes.ts / reconcile/notify.ts. Implements MOD-01/02/03 + OUT-07 for the
mixed-subject family. catalog-uat + notify-v2 stay green.
</objective>
