**Edit 1 -- collapse the state ternary and drop the parameter (D-83.1-03).**

```typescript
function renderSkillLegend(
  legend: readonly SkillLegendEntry[] | undefined,
  inheritSkills: boolean,
): string {
  if (legend === undefined || legend.length === 0) {
    return "";
  }

  const notPreloadedAnnotation = inheritSkills
    ? "available on demand"
    : "not available in this session";
  const entryLines = legend.map((entry) => {
    const annotation = entry.preloaded ? "preloaded in your context" : notPreloadedAnnotation;
    return `- \`${entry.token}\` → skill \`${entry.generatedName}\` (${annotation})`;
  });
```

"available on demand"`; remove the `inheritSkills` parameter (private function,

inside test expected-constants via the same escape). BSD grep skips
glyph-bearing files as binary -- all verification greps use `-a`.
