## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Reducer → executor | The reducer awaits `activeExecutor` (production: `dispatchHookExec`; tests: a spy). The executor's return type is `Promise<HookExecResult>` — untrusted in the sense that a buggy executor could violate the discriminated-union contract (e.g., return a result with a `kind` outside the 4-arm union). |
| Adapter → Pi handler return | The adapter converts `HookExecResult` to a Pi-event-specific return shape. A bug in the adapter could send a malformed value to Pi's runtime. |
| Mutate composition | Entry N's `result.updatedInput` is applied to `event.input` in place before entry N+1 runs. A malicious or buggy hook could mutate `event.input` into a shape that crashes a subsequent entry. |
