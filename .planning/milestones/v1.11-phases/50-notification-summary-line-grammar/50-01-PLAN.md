<objective>
Fix the v1.10 notification-grammar violation: standalone `marketplace-not-added`
and failed `plugin-info` notifications emit body-only, so the host `Error:` label
glues directly onto the detail row (`Error: ⊘ y [user] (failed) {not added}`) with
no summary. Route every error/warning emission -- standalone AND cascade -- through
ONE shared summary-prepend helper, and extend `buildSummaryLine` to return the
failed-subject summary for the two standalone error kinds. Rewrite the ~9 catalog
fence bodies + 8 prose sentences that encoded the violation as the contract, update
the existing standalone byte test, add a failed `plugin-info` byte test, and add a
cross-cutting grammar-invariant test. Closes GRAM-01..05.

Purpose: Every error/warning notification carries a non-empty summary line with the
detail rendered as its own separate block below -- emitted through a single shared
path so the standalone-vs-cascade divergence that caused the defect cannot recur
(GRAM-04). This is the operator's prior captured label-grammar preference (MEMORY).

Output: Corrected `shared/notify.ts` emission seam + `buildSummaryLine` extension,
corrected `docs/output-catalog.md` byte contract, updated/extended `notify-v2`
byte tests, a new `notify-grammar-invariant` cross-cutting gate, and a GREEN
`npm run check`.

ATOMICITY (NON-NEGOTIABLE): the `catalog-uat` byte-equality gate compares `notify()`
output against the catalog fence bodies. A code-only change (prepending the summary)
while the fences still show the glued single line makes `catalog-uat` go RED, and a
catalog-only change while the code still emits body-only also goes RED. Therefore the
notify.ts fix, the `buildSummaryLine` extension, the catalog fence + prose rewrites,
the updated/new notify-v2 tests, and the new invariant test ALL land in ONE plan and
commit together. Do NOT split tests-first into a separate committed wave -- that would
leave the suite RED across a commit boundary. The tasks below are authored in
test-first order WITHIN one plan but verified and committed as a single GREEN unit.
